## 2. 공정내용 CSV 붙여넣기 탭

### 기능
- 5번 공정내용 섹션에 **직접입력 / CSV 붙여넣기** 2개 탭 추가
- CSV 헤더 자동 인식: `공정, 기인물, 위험요인, 개선대책`
- 반영 대상:
  - **공정내용 (U7)**: 공정명 + 기인물 자동 추가
  - **위험성평가 (AB7)**: 작업단계 + 유해위험요인 + 안전조치사항 자동 행 생성

### CSV 데이터 예시
```
순번,공정,기인물,위험요인,개선대책,개선대책 근거,SIF 사고사망사례
1,작업허가서 발행 및 작업계획 수립,작업허가서 / 작업계획서,작업계획서 미작성...,사전조사 실시...,「산업안전보건기준에 관한 규칙」...,2017년 10월경...
```

### 변경 파일
- `js/sections.js`: `SectionCSV` 모듈 신규 (파싱, 반영 로직)
- `index.html`: 공정 섹션에 탭 UI + CSV textarea 추가

## 5. 출력 서식 PDF 양식 준용

### 대조 기준
- **서식3**: `서식/5. 위험성평가 지침서.pdf` pp.57-61
- **TBM 양식**: `서식/TBM 양식.pdf` 2페이지

### 페이지별 수정 내역

| 쪽 | 서식 | 수정 내역 |
|----|------|----------|
| 1쪽 | 작업개요 | 서식3 "작 업 개 요" 레이아웃 준용 — 구분/세부내용 2단, 유사재해·아차사고사례 분리, 필요한 자료명 추가 |
| 2쪽 | 위험성평가표 | 서식3 "위 험 성 평 가 표" 준용 — 현재 위험성/추가 개선 후 위험성 헤더 매칭 |
| 3쪽 | 감소대책 | 서식3 [참고] 준용 — 본질적(근원적)→공학적→관리적→개인보호구 순, 설명 문구 반영 |
| 4쪽 | 개선결과 | 서식3 "개 선 결 과" 준용 — 가능성/중대성/위험성 분리 표시, 개선 전 사진/개선 후 사진, 유해위험요인/개선내용 분리 |
| 5쪽 | 결과확인 | 서식3 "위험성평가 결과 확인" 준용 — **3단 30명 서명란** (1~10, 11~20, 21~30), 상단 확인 문구 2줄, 하단 안내문 |
| 6쪽 | TBM 1면 | TBM 양식.pdf 1면 준용 — 결재란(담당/관리감독자/소장), 안전허가 순서, 체크리스트 8문항(PDF 원본), 참석자 건강상태 세분 |
| 7쪽 | TBM 2면 | TBM 양식.pdf 2면 준용 — 피드백/기타 빈 서식 유지, 감시자 안전장구, 도급인 전달사항 |

### TBM 체크리스트 (PDF 원본 8문항으로 교체)
1. 작업에 필요한 안전작업허가서는 발행되었는가?
2. 작업조건과 작업범위는 검토되었는가?
3. 유해·위험요인에 대하여 교육 및 안전조치는 하였는가?
4. 작업에 필요한 안전한 작업방법은 알고 있는가?(장비, 공구사용법 등)
5. 도면 및 정비절차서는 검토하였는가?
6. 사용장비에 대한 안전확인점검 실시여부를 확인 하였는가?
7. 작업대상 설비는 계통과 격리되어 위험은 없는가?
8. 안전사고 발생시 비상조치계획을 알고 있는가?

### 변경 파일
- `js/print.js`: 전체 7쪽 재작성
- `js/sections.js`: TBM `CHECKLIST_ITEMS` 교체
